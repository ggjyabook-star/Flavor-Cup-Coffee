# Flavor Cup Coffee - Propuesta de Marketing Digital

Este repositorio contiene el código fuente para la landing page explicativa de la propuesta de marketing digital integral diseñada para **Flavor Cup Coffee**.

## Estructura del Proyecto

*   `index.html` - La estructura de la landing page con todos los módulos de servicio, simuladores y secciones informativas.
*   `styles.css` - Hoja de estilos con diseño responsive, variables CSS (esquema de color espresso/crema/oro) y efectos interactivos.
*   `script.js` - Lógica de control para el simulador de teléfono (Instagram feed), calculadora dinámica de inversión, acordeones de FAQ y enlace automatizado de WhatsApp.
*   `hero_coffee_shop.png` - Visual del héroe generado por AI para el diseño estético de la propuesta.

## Tecnologías Utilizadas

*   HTML5 Semántico
*   CSS3 (Flexbox, Grid, Variables CSS, Glassmorphism, CSS Transitions)
*   Vanilla JavaScript (ES6+, Intersection Observer para animaciones al hacer scroll)
*   Lucide Icons para iconografía moderna
